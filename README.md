# Strategic Impact of Generative AI on Retail & CPG

A BCA project exploring how Generative AI enhances personalization, optimizes supply chains, and improves customer experience in the retail and consumer packaged goods sector.

## 📁 Project Files
- **`Project Report.pdf`** : The complete 80+ page project report.
- **`Presentation.pptx`** : Final presentation slides summarizing the project.

## ⭐ Key Takeaways
- GenAI can reduce product description creation time by **85%**.
- AI-powered chatbots can cut customer service costs by **30-50%**.
- A strategic, five-phase framework for responsible AI adoption is proposed.
